# Skills de UI Design Premium

Este diretório reúne as melhores ferramentas e diretrizes para criar interfaces de usuário (UI) de nível mundial, focando em estética premium, acessibilidade e sistemas de design modernos.

## Coleção de Skills

### 1. [Frontend Design](frontend-design/)
- **Foco**: Estética de "alto artesanato" (high-craft).
- **Uso**: Para criar identidades visuais memoráveis e fugir do visual genérico de "IA". Prioriza decisões de design intencionais.

### 2. [UI/UX Pro Max](ui-ux-pro-max/)
- **Foco**: Inteligência de Design massiva.
- **Uso**: Biblioteca de referência para estilos (glassmorphism, bento grid), paletas de cores, pareamento de fontes e gráficos.

### 3. [Stitch UI Design](stitch-ui-design/)
- **Foco**: Prototipagem com Google Stitch.
- **Uso**: Guia técnico para criar prompts eficazes na ferramenta Stitch, facilitando o fluxo de design-para-código.

### 4. [Radix UI Design System](radix-ui-design-system/)
- **Foco**: Componentes acessíveis e "headless".
- **Uso**: Essencial para construir sistemas de design do zero que sejam 100% acessíveis (WAI-ARIA) sem sacrificar o controle visual.

### 5. [Tailwind Design System](tailwind-design-system/)
- **Foco**: Escalabilidade e Tokens de Design.
- **Uso**: Padrões para organizar utilitários Tailwind em um sistema de design coeso e profissional.

### 6. [UI Visual Validator](ui-visual-validator/)
- **Foco**: Controle de Qualidade Visual.
- **Uso**: Skill de verificação rigorosa para garantir que a implementação final corresponde perfeitamente ao design planejado e às regras de acessibilidade.

### 7. [Web Design Guidelines](web-design-guidelines/)
- **Foco**: Conformidade e Boas Práticas.
- **Uso**: Auditoria final baseada nas diretrizes de interface web da Vercel para garantir performance e UX.

## Como Alcançar um Design Premium

1. **Defina a Estética**: Comece pela skill **Frontend Design** para escolher um tom (ex: minimalismo luxuoso, brutalismo editorial).
2. **Escolha a Fundação**: Utilize **UI/UX Pro Max** para selecionar paletas de cores e fontes que harmonizem com o tom.
3. **Construa com Acessibilidade**: Se estiver criando componentes do zero, use a lógica do **Radix UI**.
4. **Valide Rigorosamente**: Ao finalizar, ative o **UI Visual Validator** para caçar inconsistências.
